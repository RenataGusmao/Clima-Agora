# Clima Agora
Este projeto foi desenvolvido como uma atividade prática com o objetivo de criar uma aplicação web simples utilizando HTML, CSS e JavaScript, realizando o consumo de uma API pública.

A aplicação exibe informações básicas do clima da cidade do Recife, como temperatura e dados do vento, obtidas em tempo real por meio da API Open-Meteo.

O projeto também foi desenvolvido com foco em responsividade, priorizando a visualização em dispositivos móveis.
