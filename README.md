# Clima Agora
Este projeto consiste no desenvolvimento de uma aplicação web simples e responsiva utilizando HTML, CSS e JavaScript, com o objetivo de realizar o consumo de uma API pública e apresentar os dados obtidos de forma dinâmica na interface do usuário.

A aplicação foi desenvolvida priorizando a visualização em dispositivos móveis, seguindo os princípios de design responsivo e organização de código voltados para aplicações web modernas.

Objetivo do Projeto

O objetivo deste projeto é demonstrar, de forma prática, a utilização de requisições a APIs públicas em aplicações web, possibilitando a integração entre sistemas e a exibição de informações em tempo real ao usuário.

Além disso, o projeto busca aplicar conceitos fundamentais do desenvolvimento front-end, como:

Estruturação de páginas com HTML

Estilização com CSS

Manipulação do DOM

Requisições HTTP utilizando JavaScript

Consumo de APIs públicas

Desenvolvimento de interfaces responsivas

Tecnologias Utilizadas

As seguintes tecnologias foram utilizadas no desenvolvimento da aplicação:

HTML5 – Estrutura da aplicação

CSS3 – Estilização e responsividade

JavaScript (ES6+) – Lógica da aplicação e consumo da API

Fetch API – Realização das requisições HTTP

API pública – Fonte dos dados exibidos no sistema
